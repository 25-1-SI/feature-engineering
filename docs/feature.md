# 계획

## 1. 공간(Spatial) 피처
1. 고위험 구간 포함 여부
   - 각 싱크홀 발생 지점(lat,lon)이 고위험 폴리라인(lat1–lon2)에 얼마나 근접하는지
   - 예: 버퍼(반경 50m) 내 포함 여부 또는 선까지의 최소 거리
2. 주변 인프라 밀도
   - 도로 굴착/복구 공사 횟수: 반경 500m 이내 최근 1년간 도로굴착 이력 개수
   - 일반 건설 프로젝트 수: 반경 1km 이내의 프로젝트 개수 및 총 사업금액 합계
   - 지하수 공사 밀도: 지하역류 공사 포인트 개수
3. 지하철 특성
   - 최근접역 거리: 각 지점에서 가장 가까운 지하철역까지 직선거리
   - 역 깊이: 그 역의 정거장깊이 (깊을수록 굴착·안정화 작업 영향 가능성)
4. 행정구역·지형
   - 위도·경도로 행정구역(구, 동) 매핑 → 원-핫 인코딩 or 임베딩
   - (추가) 서울시·국토지리원에서 제공하는 고도·경사도 레이어를 겹쳐서 “경사도” 피처 생성


## 2. 시간(Temporal) 피처
1. 발생 시기
   - 발생일자 → 년, 월, 계절(봄·여름·가을·겨울)
   - 예: 장마철(6~8월) 집중 발생 가능성
2. 공사 타이밍
   - 해당 위치 반경 내 가장 최근 도로·건설 공사 완료일로부터 경과일수
   - 혹은 착공일과 발생일자 간 “공사 중” 여부
3. 복구 주기
   - 동일 위치(주소)에서 과거 발생 → 복구완료 → 재발생까지 걸린 기간


## 3. 도메인(Domain-specific) 피처
1. 싱크홀 규모
   - 발생규모폭, 발생규모연장, 발생규모깊이: 모델 입력 그대로 사용하거나, 이들의 곱(부피) 혹은 주성분
   - 복구비용(복구비용) 로그 변환
2. 지질 특성
   - 발생지역지질종류명: 그룹별 더미 변수
   - (추가 데이터 연계) 지반암 암종·토질정보
3. 인간 영향
   - 피해차량대수 등 직접 피해 규모
